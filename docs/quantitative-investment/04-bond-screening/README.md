# 转债筛选系统

## 系统目标

通过量化指标筛选优质可转债，在控制风险的前提下获取"下有保底、上不封顶"的投资机会。

## 转债基础知识

### 1. 转债特性
- **债券属性**：到期还本付息，提供下行保护
- **期权属性**：可转换为正股，分享上涨收益
- **双重收益**：债券利息+转股收益
- **风险较低**：相比正股波动更小

### 2. 关键概念
- **转股价**：转债转换为正股的价格
- **转股价值**：按当前正股价计算的转债理论价值
- **转股溢价率**：转债价格相对转股价值的溢价
- **到期收益率**：持有到期的年化收益率

## 筛选指标体系

### 1. 价格指标

#### 转债价格
- **标准**：转债价格 < 110元
- **说明**：价格越低，下行风险越小
- **分档**：
  - 优秀：< 105元
  - 良好：105-110元
  - 一般：110-120元
  - 避免：> 120元

#### 转股溢价率
- **标准**：转股溢价率 < 20%
- **说明**：溢价率越低，与正股联动性越强
- **计算公式**：
```
转股溢价率 = (转债价格 - 转股价值) / 转股价值 × 100%
```

### 2. 收益指标

#### 到期收益率
- **标准**：到期收益率 > 2%
- **说明**：提供债券保底收益
- **计算方法**：考虑利息和本金偿还
- **特殊情况**：可提前赎回的转债需调整

#### 纯债价值
- **标准**：纯债价值 > 85元
- **说明**：债券属性提供的保护程度
- **计算**：按同等级债券收益率折现

### 3. 规模指标

#### 转债规模
- **标准**：转债规模 > 5亿元
- **说明**：保证流动性，避免操纵风险
- **分档**：
  - 大盘：> 20亿元
  - 中盘：5-20亿元
  - 小盘：< 5亿元（谨慎）

#### 剩余期限
- **标准**：剩余期限 > 2年
- **说明**：给转股留出足够时间
- **最优**：3-5年期限

### 4. 信用指标

#### 信用评级
- **标准**：信用评级 ≥ AA
- **说明**：降低违约风险
- **评级机构**：中诚信、联合、大公等

#### 正股基本面
- **ROE**：> 8%
- **负债率**：< 70%
- **现金流**：经营现金流为正
- **盈利稳定性**：连续盈利 > 3年

## 筛选流程

### 第一步：基础筛选
```
1. 剔除已公告赎回的转债
2. 剔除正股停牌超过3个月的转债
3. 剔除规模小于5亿的转债
4. 剔除信用评级低于AA的转债
```

### 第二步：价格筛选
```
1. 转债价格 < 110元
2. 转股溢价率 < 20%（可转股期内）
3. 到期收益率 > 2%
4. 纯债价值 > 85元
```

### 第三步：正股筛选
```
1. 正股ROE > 8%
2. 正股负债率 < 70%
3. 正股连续盈利 > 3年
4. 正股现金流稳定
```

### 第四步：综合排序
```
1. 按转债价格从低到高排序
2. 综合考虑到期收益率
3. 重点关注正股质量
4. 优选大规模转债
```

## 转债分类策略

### 1. 防御型转债
#### 特征
- 转债价格 < 105元
- 到期收益率 > 3%
- 纯债价值 > 90元
- 正股为稳定行业

#### 投资策略
- 重点关注债券属性
- 适合熊市或震荡市
- 持有到期或转股
- 仓位可适当集中

### 2. 进攻型转债
#### 特征
- 转股溢价率 < 10%
- 正股成长性好
- 正股估值合理
- 行业景气度上升

#### 投资策略
- 重点关注转股价值
- 适合牛市或成长期
- 及时转股或卖出
- 仓位需要分散

### 3. 平衡型转债
#### 特征
- 价格在105-110元
- 溢价率10-20%
- 到期收益率2-4%
- 正股质地中等

#### 投资策略
- 攻守兼备
- 适合大部分市场环境
- 灵活调整策略
- 核心配置品种

### 计算公式
#### 转股价值
```excel
=正股价格/转股价*100
```

#### 转股溢价率
```excel
=(转债价格-转股价值)/转股价值
```

#### 综合评分
```excel
=价格得分*0.3+收益得分*0.3+正股得分*0.4
```

## 投资策略

### 1. 买入策略
#### 分批买入
```
价格 < 100元：买入40%目标仓位
价格 < 95元：买入60%目标仓位
价格 < 90元：买入80%目标仓位
价格 < 85元：买入100%目标仓位
```

#### 时机选择
- 新债上市首日（破发时）
- 正股大幅下跌时
- 市场恐慌时期
- 转债到期前1年

### 2. 卖出策略
#### 价格卖出
```
价格 > 130元：卖出30%
价格 > 140元：卖出50%
价格 > 150元：卖出70%
价格 > 160元：全部卖出
```

#### 转股卖出
```
转股溢价率 < 5%：考虑转股
正股涨幅 > 30%：转股后卖出正股
强制赎回公告：立即转股
```

### 3. 持有策略
#### 长期持有
- 优质转债可长期持有
- 关注正股基本面变化
- 定期评估转债价值
- 适时调整仓位

#### 波段操作
- 利用价格波动获利
- 高抛低吸策略
- 控制交易频率
- 注意交易成本

## 风险控制

### 1. 集中度控制
- 单只转债仓位 < 5%
- 同行业转债仓位 < 15%
- 转债总仓位 < 30%

### 2. 流动性控制
- 优选大规模转债
- 避免冷门转债
- 分批建仓减仓

### 3. 信用风险控制
- 严格筛选信用评级
- 关注正股财务状况
- 分散投资降低风险

## 常见问题

**Q：转债价格为什么会破发？**
A：新债上市时市场情绪、正股表现、供需关系等因素影响。

**Q：什么时候应该转股？**
A：转股溢价率很低（<5%）且看好正股前景时。

**Q：如何应对强制赎回？**
A：及时转股，避免损失。

**Q：转债投资的主要风险是什么？**
A：正股下跌、利率上升、信用风险、流动性风险。
