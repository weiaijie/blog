# 医院教育管理系统 - 项目业务包详细报告

## 📊 项目规模总览

- **业务模块总数**: 100个一级业务模块
- **目录总数**: 1020个业务目录
- **Vue文件总数**: 2309个Vue组件文件
- **技术栈**: Vue 2.6.14 + Element UI + Vuex + Vue Router
- **开发模式**: 大型团队协作开发（15+开发者）
