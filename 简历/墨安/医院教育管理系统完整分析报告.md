# 医院教育管理系统 (hoe_vue) 完整深度分析报告

## 📋 项目概览

### 基本信息
- **项目名称**: hoe_vue (Hospital Education Management System)
- **项目类型**: 超大型医院教育培训管理平台前端系统
- **技术栈**: Vue 2.6.14 + Element UI + Vuex + Vue Router
- **开发团队**: 多人协作开发 (15+ 开发人员)
- **当前版本**: 4.4.0
- **开发环境**: Node.js + Vue CLI 4.4.4
- **路由架构**: 前后端混合路由 (大部分路由配置存储在后端)

### 项目规模统计
- **代码行数**: 约100万行+ (包含注释和配置)
- **文件数量**: 3500+ 文件
- **功能模块**: 100+ 主要业务模块
- **API接口**: 1000+ 接口
- **组件数量**: 600+ 自定义组件
- **页面数量**: 1500+ 业务页面
- **业务角色**: 25+ 不同用户角色
